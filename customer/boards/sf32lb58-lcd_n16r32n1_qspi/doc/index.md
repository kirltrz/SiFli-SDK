# SF32LB58-LCD_N16R32N1_QSPI
`sf32lb58-lcd_n16r32n1_qspi` board is based on [SF32LB58-DevKit-LCD](https://wiki.sifli.com/board/sf32lb58x/SF32LB58-DevKit-LCD.html) board and 
has module [SF32LB58-MOD-N16R32N1](https://wiki.sifli.com/silicon/%E6%A8%A1%E7%BB%84%E5%9E%8B%E5%8F%B7%E6%8C%87%E5%8D%97.html#sf32lb58-mod) on the board. 
`1.85 rect QSPI Single-Screen LCD(LCD_1P85_390*450_DevKit_Adapter_V1.0.0)` with QSPI interface is used as default display.
